You should find out how many disconnected subnetworks you have after crushing several nodes in the network

Input: Two arguments. Network structure (as list of connections between nodes) and list of crashed nodes

Output: Int. The amount of subnetworks after crushing nodes.

Example: 

subnetworks([
        ['A', 'B'],
        ['B', 'C'],
        ['C', 'D']
    ], ['B']) == 2
subnetworks([
        ['A', 'B'],
        ['A', 'C'],
        ['A', 'D'],
        ['D', 'F']
    ], ['A']) == 3
subnetworks([
        ['A', 'B'],
        ['B', 'C'],
        ['C', 'D']
    ], ['C', 'D']) == 1
